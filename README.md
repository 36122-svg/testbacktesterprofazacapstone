# Crypto Backtester Pro

🚀 Deploy ke Netlify dengan drag-and-drop!

## Cara Deploy (Tanpa Terminal)

1. **Upload ke GitHub** (pakai GitHub Desktop, drag folder ini ke GitHub Desktop).
2. **Buka Netlify** → Klik "Add new site" → "Import an existing project" → Pilih GitHub.
3. **Build settings otomatis terdeteksi** (atau isi: Build Command = `npm run build`, Publish Directory = `dist`).
4. Klik Deploy!
